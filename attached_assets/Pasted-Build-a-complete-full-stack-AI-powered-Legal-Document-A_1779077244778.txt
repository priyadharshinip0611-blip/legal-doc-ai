Build a complete full-stack AI-powered Legal Document Analyzer web application using React, Vite, Tailwind CSS, Node.js/Express, and OpenAI API.

The application should look modern, professional, and production-ready with a premium legal-tech design theme using dark blue, white, slate, and gold accent colors.

==================================================
TECH STACK
==================================================

Frontend:
- React + Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Lucide React icons

Backend:
- Node.js + Express

AI:
- OpenAI GPT-4o-mini API

Database:
- Firebase Firestore or MongoDB

File Handling:
- PDF upload support
- Image upload support (PNG/JPG/JPEG)
- PDF text extraction

3. AI Legal Analysis
When user uploads a legal document:
- Extract text from PDF or image
- Send extracted text to OpenAI
- AI should return structured JSON with:
  - parties involved
  - key dates
  - clause summaries
  - extracted entities
  - risks
  - overall summary

Use this AI prompt internally:

"You are a legal AI assistant. Analyze this legal document and provide a structured summary.

Extract:
1. All parties involved
2. Key dates
3. Clause summaries
4. Obligations
5. Penalties
6. Risks
7. Important legal entities
8. Overall executive summary

Use simple language but preserve legal meaning."
